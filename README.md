Secure P2P WirelessLAN file transmition util

Based on UDP broadcast and device detection, connection establishing with mutual signing
Passphrase-based authorization, MITM proof

Params:
--fresh(-f) - clear all application data, cert, cache
--passphrase(-p) - define passphrase for devices authorization

Menu based CLI
